Virtual Host Testing

This page demonstrates that virtual hosting is working correctly for naruto.com.

Technical Details:

  • Hostname: naruto.com
  • Port: 3000
  • Document Root: ./www-vhosts/naruto.com
  • Server Block: Second server in eval.conf

What This Tests:

When you access this page via curl --resolve naruto.com:3000:127.0.0.1 http://naruto.com:3000/about.html, the WebServ server uses the Host header to determine which server block should handle the request.

The Config::findServer() method matches:

  1. Port number (3000)
  2. Server name (naruto.com)
  3. Returns the appropriate ServerConfig

Served by WebServ - Virtual Host: naruto.com - Page: /about.html
