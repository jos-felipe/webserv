Virtual Host Validation

This page proves that sasuke.com serves different content than other virtual hosts.

Configuration Details:

  • Hostname: sasuke.com
  • Port: 3000
  • Document Root: ./www-vhosts/sasuke.com
  • Server Block: Third server in eval.conf

Testing Methodology:

Access via: curl --resolve sasuke.com:3000:127.0.0.1 http://sasuke.com:3000/team.html

This tests the server's ability to:

  • Handle multiple virtual hosts on the same port
  • Serve distinct content trees per hostname
  • Properly route requests based on Host header
  • Maintain separate configurations per virtual host

Expected Behavior:

Each virtual host should return its own unique content, demonstrating that the Config::findServer() method correctly identifies and routes requests to the appropriate server configuration.
