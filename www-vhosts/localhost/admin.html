Default Virtual Host

This page is served from the localhost default virtual host configuration.

Server Configuration:

  • Hostname: localhost
  • Port: 8080
  • Document Root: ./www-vhosts/localhost
  • Server Block: First server in eval.conf

Testing Access:

Access via: curl http://localhost:8080/admin.html

Or: curl --resolve localhost:8080:127.0.0.1 http://localhost:8080/admin.html

Default Host Purpose:

This virtual host serves as the fallback when:

  • No specific server_name matches the request
  • Direct IP access is used
  • Invalid or missing Host headers
  • Default administrative access is needed

Virtual Host Resolution:

The server should properly distinguish between requests for different hostnames even when they resolve to the same IP address, demonstrating proper virtual hosting implementation.
